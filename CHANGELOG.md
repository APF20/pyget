# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.3.2] - 2026-03-20

### Modified

- Updated minimum version requirement to python >=3.9.
- CI: changed from travisCI to Github Actions.

## [3.2.2] - 2026-03-17

### Changelog Update (2026-03-19)

- Tests: added unittests for pyget REST and WebSocket APIs (tests/test_pyget.py).

### Added

- The `pyget` module.
- MANIFEST, README, LICENSE, and CHANGELOG files.
- Example files for REST and WebSocket APIs.

## [1.0.0 ~ 3.2.2] - 2021-12-08
